# 10 Academy AI Mastery – Week 1 Challenge

## Financial News Sentiment Analysis

### Project Overview
This project analyzes financial news headlines and explores their relationship with stock market movements. The objective is to apply exploratory data analysis (EDA), natural language processing (NLP), sentiment analysis, and financial indicators to identify patterns that may help predict stock price behavior.

## Tasks

### Task 1 – Exploratory Data Analysis
- Data loading and cleaning
- Headline statistics
- Publisher analysis
- Publication trend analysis
- Keyword extraction
- Topic modeling using LDA
- Data visualizations

### Task 2 – Quantitative Analysis
- Technical indicators
- Moving averages
- RSI
- MACD
- Financial visualization

### Task 3 – Correlation Analysis
- Sentiment scoring
- Daily return calculation
- Correlation between news sentiment and stock movement
